11 5 razones para aprender Ruby

  1. La sintaxis.
  2. La sintaxis es muy amigable y se lee casi como lenguaje natural. El performance no es igual que en lenguajes de más bajo nivel....
  3. Ruby on Rails.
  4. Ruby es muy popular en aplicaciones web porque existe Ruby on Rails un framework muy popular para hacer aplicaciones web que algún día utilizaron o siguen utilizando sitios como Twitter, Github, Airnbn, SoundCloud...
  5. Java está en todas partes.
  6. Se estima que Java se ejecuta en más de 15.000.000.000 de dispositivos en todo el mundo (sí, 15 mil millones, has leído bien)...
  7. Orientado a objetos.
  8. Ruby es un lenguaje orientado a objetos, como JAVA, C++, Python, etc. Con la característica de que Ruby se toma muy en serio su papel sobre ser orientado a objetos. En Ruby, por ejemplo, no hay primitivos, como int, todo es un objeto, incluso los operadores como +, < son objetos.....
  9. ¿Es esto interesante?.
  10. Personalmente lo es, por ejemplo puedes ejecutar métodos sobre lo que en otros lenguajes serían primitivos, algo así como convertir un número a cadena: numero = 1 = numero.to_s.
