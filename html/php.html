6 5 razones para aprender Php

1. Es económico:
Es económico… gratis realmente, PHP es la prueba (o excepción) de que lo gratis no necesariamente es de peor calidad que lo pagado.

2. Muy fácil:
PHP contiene muchas características y funciones especiales necesarias para crear páginas web dinámicas. Y como ya lo mencionábamos antes, el lenguaje PHP está diseñado para incluirse fácilmente en un archivo HTML.

3. Multisistema:
Se puede ejecutar en muchos sistemas operativos distintos; Windows, Linux, Mac OS, y la mayoría de las variedades de Unix.

4. Seguridad:
El usuario no puede ver el código PHP.

5. Es personalizable:
Su licencia de código abierto permite a los programadores modificar el software PHP, agregando o modificando sus características según se requiera para ajustarse a los ambientes específicos de cada uno.
