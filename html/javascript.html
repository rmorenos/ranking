7 Razones para aprender JavaScript

Razones para aprender JavaScript

1. Es el lenguaje de programación de los navegadores web:
Todos los más importantes lo soportan y lo tienen activado por defecto: Firefox, Chrome, IE, Opera, Safari, lo que lo convierten en el lenguaje más popular en Internet.

2. Está totalmente integrado:
HTML y CSS. Cada uno juega un papel definido en una página web: HTML es el lenguaje de marcado para crear la estructura, CSS el lenguaje de estilos para configurar la presentación y JavaScript le proporciona interactividad y comportamiento dinámico.

3. Es muy potente y expresivo:
Con sintaxis que guarda similitudes con otros lenguajes muy populares, pero con características particulares.

4. Fácil de aprender:
Es fácil de aprender. Tienen una curva de aprendizaje rápida y exige poca inversión de tiempo para empezar a utilizarlo.
